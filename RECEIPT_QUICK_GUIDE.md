# Quick Guide: Printing Transaction Receipts

## FEATURE READY TO USE!

You can now print professional receipts for all payments and expenses!

---

## How to Print a Receipt

### Method 1: From Payments Dashboard
```
1. Login: accounts / accounts123
2. Go to: Payments (navbar)
3. Find any transaction
4. Click: "Print" button (printer icon)
5. Receipt opens in new window
6. Click: "Print Receipt" button
7. Print dialog appears → Print!
```

### Method 2: From Withdrawals Dashboard
```
1. Login: accounts / accounts123
2. Go to: Withdrawals (navbar)
3. Find any expense
4. Click: "Print" button (printer icon)
5. Receipt opens in new window
6. Click: "Print Receipt" button
7. Print dialog appears → Print!
```

## What's on the Receipt

### For Payment Receipts:
- School name and logo
- Receipt number (unique ID)
- Payment date
- **Student full name**
- **Student registration date**
- Payment type (Registration, School Fee, etc.)
- Amount paid (green, positive)
- Description/notes
- Thank you message
- Issued by and print date

### For Expense Receipts:
- School name and logo
- Receipt number
- Expense date
- **Expense description** (NO student info)
- Amount withdrawn (red, negative)
- Expense type
- Official record notice
- Issued by and print date

## Pro Tips

### Tip 1: Print Multiple Receipts
- Right-click "Print" button → Open in new tab
- Print each in separate window
- All receipts ready at once!

### Tip 2: Save as PDF
- Click Print Receipt
- In print dialog, select "Save as PDF"
- Great for digital records!

### Tip 3: Quick Print
- After receipt opens, just press `Ctrl+P`
- No need to click Print Receipt button

### Tip 4: Check Before Printing
- Preview shows exactly what will print
- Verify all details are correct
- Cancel if anything needs editing
